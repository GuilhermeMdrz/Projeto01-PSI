O sistema de auxílio aos estudos é uma aplicação web que visa ajudar na organização e gerenciamento de demandas dos estudantes, de forma simples e eficaz.
Para executar a aplicação é necessário seguir os seguintes passos:
1 - Baixar as extensões:
    Vá na área de extensões do VS code e instale:
        1° Python 
        2° Português (opcional)
2 - Instale flask:
    Abra o terminal e nele execute:
        1° pip install flask
3 - Crie o ambiente virtual:
    Também no terminal, execute:
        1° python -m venv env
        2° env\Scripts\activate
    O primeiro vai criar e o segundo ativar
4 - Rode o código:
    Ainda no terminal, execute:
        1° python app.py
    Depois disso irá aparecer uma mensagem semelhante a essa:
            *Serving Flask app 'app'
            * Debug mode: on
        WARNING: This is a development server. Do not use it in a production deployment. Use a production WSGI server instead.
            * Running on http://127.0.0.1:5000
        Press CTRL+C to quit
            * Restarting with stat
            * Debugger is active!
            * Debugger PIN: 657-820-538
    Agora aperte no link que apareceu:
        EX: http://127.0.0.1:5000
Pronto! Você acessou a aplicação.
Agora aproveite as funcionalidades ofertadas de forma que desejar para facilitar seus estudos, bom uso!
